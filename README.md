# Fifo - First in, first out
Sistema gerenciador de filas e reservas de restaurantes.

# API 

A API está sendo desenvolvido usando Spring MVC com as seguintes ferramentas:

- Spring Boot
- Spring Data
- Maven
- PostgreSQL
- Hibernate
- Spring Tool Suite(IDE)
